HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use the HTML Color Codes website to get the Hexadecimal code to define the desired colour.
  • using a . before the class name will designate it as a class selector. Any HTML element assigned to that class will gain the property assignments from that CSS.
  • A link: element creates the connection to an external file.
  • The rel: attribute specifies the relationship between the current document and the linked document resource.
  • The href: attribute specifies the location (URL) of the external resource.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration (color: blue;) includes the CSS property (color) and the value for that property, (blue).
  • CSS Box Model: consists of a Margin, Border, Padding, and Content. The 'Margin' property indicates how much space we want around the outside of an element. The 'Border' is set between the 'Margin' and the 'Padding'. The 'Padding' property adds space around the content inside an element, increasing the 'Padding' makes the entire element bigger.

Git

  • ls: shows a list of files and directories on our computer.
  • cd filename: allows us to change directories using cd followed by the name of the directory.
  • pwd: short for 'Print Working Directory' allows us to check if we are in the correct directory.
  • cd ..: will allow us to navigate backwards one directory at a time.
  • mkdir name: will create a new directory in the current directory labelled as the specified 'name'.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: is used to commit all files in the working branch.
  • git commit -m "insert message": is used commit the changes to make the git repository up to date with our working directory.
  • git pull origin main: used to pull the base branch into the feature branch.
  • git push origin feature/name: is used to push up all of our changes to the repository on GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript reads the code from top to bottom.
  • The for loop is an iterative statement which you use to check for certain conditions and then repeatedly execute a block of code as long as those conditions are met.
  • The Math namespace object contains static properties and methods for mathematical constants and functions.
  • A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
